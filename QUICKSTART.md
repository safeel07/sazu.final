# 🚀 Quick Start Guide

## Get Your Birthday Surprise Running in 5 Minutes!

### Step 1: Install Node.js (if not installed)
Download and install from: https://nodejs.org/

### Step 2: Install Backend Dependencies
```bash
cd backend
npm install
```

### Step 3: Start the Backend Server
```bash
npm start
```

You should see: "🎉 Birthday Surprise Server is running!"

### Step 4: Open the Frontend
Open `frontend/index.html` in your web browser

### Step 5: Customize
1. **Set Birthday Date:** Edit `frontend/script.js` line 14
   ```javascript
   const BIRTHDAY_DATE = '2026-04-15'; // Your wife's birthday
   ```

2. **Customize Love Letter:** Edit `frontend/script.js` starting at line 20
   ```javascript
   const LOVE_LETTER = `Your message here...`;
   ```

3. **Add Photos:** See PHOTO_GUIDE.md

4. **Add Music:** See MUSIC_GUIDE.md

### That's It! 🎉

Your romantic birthday surprise is ready!
